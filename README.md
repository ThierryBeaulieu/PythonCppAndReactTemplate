## Installation

If you are on mac or linux, you should use the command:
```bat
PythonCppAndReactTemplate> npm install
```

Now that this is done âœ”ï¸, Flask's server will need dynamic libraries generated by C++ . You will need to generate those ðŸ©¹. For that, you will need to install [CMake](https://cmake.org/install/). For the beginners, CMake can be used to generate Visual Studio solutions (Windows) and projects for XCode (Mac). You can generate and build your project from the platform that you want. I recommend you to download the CMake GUI if possible (it makes things that much easier!).

If you are using the CMake GUI, you need to select as a root ðŸŒ³ :

```bat
PythonCppAndReactTemplate/
```

After that, you need to specify the location of the build folder ðŸ—ï¸ (this can be anywhere on your computer, but I recommend you keep it somewhat close, as this is where the generated files will end up). Let's say:

```bat
PythonCppAndReactTemplate/build/
```
The dynamic libraries are in the folder ```build/server/cpplib/Debug```. There should be .exp, lib and .pyd files. Take those and manually place them in the folder ```PythonCppAndReactTemplate/build/server/cppBin/```. 

You should be good to go!

## Running the web application

To activate the server, use the command:

```bat
PythonCppAndReactTemplate> npm run server
```

To activate the client, user the command:

```bat
PythonCppAndReactTemplate> npm run client
```
You should be good to go!
